# Projeto Frequentar - Protótipo

Sistema simples de CRUD de alunos utilizando Node.js, Express e React, com persistência em JSON.

## Cores
Baseadas no padrão SENAC:
- Azul: #004587
- Laranja: #F68B1F

## Pré-requisitos
- Node.js instalado.

## Instalação e Execução

### 1. Iniciar o Backend
Abra um terminal na pasta `backend`:
```bash
npm install
node server.js
